## テンプレート文字列

**文字列と変数を同時に用いるには、以下のような書き方がある**

```
const name = "お猿";
const age = 23;

const message1 = "私の名前は" + name "です。年齢は" + age + "です。";

console.log(message1);
```

出力結果

```
「私の名前はお猿です。年齢は23歳です。」
```

**テンプレート文字列は、以下のように記述する**

```
const message2 = `私の名前は${name}です。年齢は${age}です。`;
console.log(message1);
```

出力結果

```
「私の名前はお猿です。年齢は23歳です。」
```
